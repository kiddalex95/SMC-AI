# Mofenyi A Mosadi Forex Dashboard

**Live AI-powered Forex trading dashboard** built with HTML, Tailwind CSS, and JavaScript.  
Fully integrated with **TwelveData API** (live prices & candles) and **Finnhub API** (live news/events).  
Supports **signals, top-down analysis, trade journal, market strength speedometer, and alerts** — ready for deployment on Vercel.

---

## Features

- **Live Forex Prices** via TwelveData API
- **AI Top-Down Analysis** (1D → M15) with trend calculation
- **Trade Signals** for scalp, swing, and position trades
- **Open Orders Panel** with dynamic updates
- **Market Strength Speedometer**
- **Trade Journal** with persistent storage
- **News & Events** via Finnhub API (updates every 60s)
- **Responsive and futuristic design** with Tailwind CSS
- **Automatic refresh** every 30 seconds for real-time accuracy
- **Alerts & Notifications** integrated
